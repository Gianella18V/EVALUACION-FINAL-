# Aroma — Sitio web de cafetería

Sitio web de 5 páginas para la cafetería de especialidad **Aroma**, desarrollado como
proyecto de la evaluación integral del curso de Diseño y Desarrollo Web.

## Estructura del proyecto

```
aroma-cafeteria/
├── index.html          # Inicio
├── nosotros.html        # Nosotros
├── menu.html            # Menú / Productos
├── servicios.html        # Servicios
├── contacto.html         # Contacto
├── css/
│   └── style.css        # Estilos propios del sitio
├── js/
│   └── script.js         # AOS, Swiper, navbar activo, filtro de menú, formulario
└── img/                  # (imágenes propias, si se agregan)
```

## Herramientas utilizadas

- **Bootstrap 5.3** — sistema de grillas, componentes y utilidades.
- **Font Awesome 6** — iconografía.
- **Google Fonts** — Fraunces (títulos), Work Sans (texto), Space Mono (precios y etiquetas).
- **AOS (Animate On Scroll)** — animaciones al desplazarse por la página.
- **Swiper JS** — carrusel de imágenes en la página de Inicio.

## Páginas

1. **Inicio** — hero, destacados del menú, galería y testimonios.
2. **Nosotros** — historia, misión, visión, valores, línea de tiempo y equipo.
3. **Menú** — carta completa con filtro por categoría (bebidas calientes, frías, postres, snacks).
4. **Servicios** — delivery, eventos privados, catering, coworking, venta de grano y reservas.
5. **Contacto** — formulario de contacto, mapa y datos de ubicación/horario.
